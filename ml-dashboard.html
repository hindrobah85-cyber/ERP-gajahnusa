üß† ML Model Architecture

1. Payment Fraud Detection Model

Algorithm: Random Forest Classifier (100 estimators)
Model menganalisis pola pembayaran dengan features: amount, payment method, late hours, salesman behavior patterns, time factors. Menggunakan ensemble learning untuk meningkatkan akurasi deteksi fraud.

2. Anomaly Detection System

Algorithm: Isolation Forest
Unsupervised learning untuk mendeteksi anomali dalam transaksi. Contamination rate 10%, efektif menangkap outliers dan unusual patterns tanpa memerlukan labeled fraud data.

3. Demand Forecasting Model

Algorithm: Gradient Boosting Regressor / LSTM
Time series prediction dengan 7-day sequence length. Model mempelajari seasonal patterns, trends, dan historical demand untuk prediksi multi-day. Includes confidence intervals.

4. Route Optimization Engine

Algorithm: Nearest Neighbor Heuristic (TSP)
Solves traveling salesman problem menggunakan nearest neighbor dengan Haversine distance. Optimizes untuk minimum distance, time, dan fuel cost. Handles capacity constraints.

5. Behavioral Analysis Model

Algorithm: Pattern Recognition & Statistical Analysis
Menganalisis user behavior patterns: transaction frequency, timing, location consistency, payment methods. Generates risk scores dan identifies suspicious behaviors.
