🧪 Testing Guide - Fitur Baru

1. Test KTP Photo Options:

  • Pilih "📁 Upload File" → Upload gambar KTP
  • Pilih "📷 Take Photo" → Gunakan kamera langsung
  • Lihat preview hasil capture

2. Test AI Face Recognition:

  • Klik "🧠 Start AI Face Learning"
  • Allow camera permissions
  • Klik "📸 Capture & Analyze"
  • Tunggu ML processing (3 detik)
  • Lihat "✅ Face Recognition Complete"

3. Test Area Phone Logic:

  • Pilih role: Sales/Supervisor/Manager → Area Phone muncul
  • Pilih role: Driver/Gudang/Admin → Area Phone hilang

4. Test Role-based Deletion:

  • Buka Employee Management
  • Set role sebagai Admin/Owner → Delete button aktif
  • Set role lain → Delete button disabled

⚠️ Penting:

  • Gunakan Google Chrome untuk kompatibilitas terbaik
  • Allow camera permissions untuk fitur face recognition
  • Pastikan kedua server (8080 & 8002) berjalan
  • Tes semua fitur baru yang telah diimplementasi
